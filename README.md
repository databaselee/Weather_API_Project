# WeatherApp

This Python application fetches weather data from OpenWeatherMap API.

## Setup Instructions

1. Install Python 3.8+
2. Install required packages: `requests`
3. Get a free API key from https://openweathermap.org/api
4. Add your API key to the script or as an environment variable

## Usage

Run the script and enter a city name to get current weather data.
